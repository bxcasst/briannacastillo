Proyectos

  • DriveDynamics

    DriveDynamics es un sistema de gestión de flotas desarrollado para una tienda de muebles, con el objetivo de registrar y monitorear el uso de vehículos de entrega, cobradores y unidades prestadas. Como analista de requerimientos, participé en la recopilación, documentación y validación de especificaciones funcionales, asegurando que el sistema cumpliera con las necesidades del negocio

  • Programa de sensores en arduino

    Esta es una aplicación para procesar, analizar y visualizar datos de sensores en un microcontrolador Arduino, permitiendo el monitoreo en tiempo real de temperatura y humedad a través de una plataforma web con gráficos y estadísticas. Además, incorpora un sistema de alertas críticas para evitar interferencias en las lecturas. Este proyecto me permitió fortalecer habilidades en redacción técnica, documentación de software y estructuración de información para garantizar la claridad y funcionalidad del sistema.
